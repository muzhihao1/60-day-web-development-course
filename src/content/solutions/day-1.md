# Day 1 解决方案：Git基础

## 解决方案步骤

1. **初始化Git仓库**
   ```bash
   git init
   ```

2. **创建README.md文件**
   ```bash
   echo "# My Project" > README.md
   ```

3. **添加文件到暂存区**
   ```bash
   git add README.md
   ```

4. **提交更改**
   ```bash
   git commit -m "Initial commit: Add README"
   ```
