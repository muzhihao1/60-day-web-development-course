ARIA属性使用示例

本页面展示了常见的ARIA属性及其正确使用方法。

1. aria-label - 提供可访问的名称

当视觉元素没有文本内容时，使用 aria-label 提供描述。

<button aria-label="关闭对话框">&times;</button>
<button aria-label="搜索">🔍</button>
<button aria-label="添加到收藏夹">⭐</button>

2. aria-labelledby - 关联已有标签

使用 aria-labelledby 关联已存在的文本作为标签。

账单地址


<h3 id="billing">账单地址</h3>
<div role="group" aria-labelledby="billing">
    <label for="billing-name">姓名：</label>
    <input type="text" id="billing-name">
    ...
</div>

3. aria-describedby - 提供额外描述

使用 aria-describedby 关联帮助文本或错误信息。

密码必须包含至少8个字符，包括大小写字母和数字 
<input type="password" id="password" 
       aria-describedby="pwd-help pwd-error">
<span id="pwd-help">密码要求说明...</span>
<span id="pwd-error" role="alert"></span>

4. aria-expanded 和 aria-controls - 控制展开/折叠

用于下拉菜单、手风琴等可展开/折叠的组件。

  • 选项 1
  • 选项 2
  • 选项 3
<button aria-expanded="false" 
        aria-controls="dropdown-menu"
        aria-haspopup="true">
    选择选项 ▼
</button>
<div id="dropdown-menu" aria-hidden="true" role="menu">
    ...
</div>

5. role 和 aria-current - 定义角色和当前状态

使用 role 定义元素角色，aria-current 标识当前项。

  1. 首页 /
  2. 产品 /
  3. 笔记本电脑
<nav role="navigation" aria-label="面包屑">
    <ol>
        <li><a href="/">首页</a></li>
        <li><a href="/products">产品</a></li>
        <li><a href="/products/laptops" 
               aria-current="page">笔记本电脑</a></li>
    </ol>
</nav>

6. aria-live - 动态内容通知

使用 aria-live 通知屏幕阅读器内容的动态变化。

<!-- 礼貌通知（等当前朗读完成） -->
<div aria-live="polite" aria-atomic="true"></div>

<!-- 紧急通知（立即朗读） -->
<div role="status" aria-live="assertive"></div>

7. 表格可访问性

使用适当的ARIA属性增强表格可访问性。

2024年第一季度销售报告
月份 销售额 增长率
一月 ¥50,000 +15%
二月 ¥45,000 +10%
三月 ¥60,000 +20% 
<table role="table" aria-label="2024年销售数据">
    <caption>2024年第一季度销售报告</caption>
    <thead>
        <tr>
            <th scope="col">月份</th>
            <th scope="col">销售额</th>
            <th scope="col">增长率</th>
        </tr>
    </thead>
    ...
</table>

9. 进度和加载状态

使用ARIA属性表示进度和加载状态。

上传进度：60%

正在加载数据，请稍候...

<div role="progressbar" 
     aria-valuenow="60" 
     aria-valuemin="0" 
     aria-valuemax="100"
     aria-label="上传进度">
    ...
</div>

10. 表单验证和错误处理

正确标记必填字段和错误信息。

请输入有效的邮箱地址 
<label for="email">
    邮箱地址 <span aria-label="必填">*</span>
</label>
<input type="email" 
       aria-required="true"
       aria-invalid="false"
       aria-describedby="email-error">
<span id="email-error" role="alert" aria-live="assertive">
    错误信息
</span>
