1. 响应式图片技术

使用 srcset 和 sizes

根据视口宽度自动选择合适尺寸
<img src="default.jpg"
     srcset="small.jpg 400w,
             medium.jpg 800w,
             large.jpg 1200w"
     sizes="(max-width: 400px) 100vw,
            (max-width: 800px) 50vw,
            33vw"
     alt="响应式图片">

使用 picture 元素（艺术指导）

<picture>
    <source media="(max-width: 600px)" srcset="mobile.jpg">
    <source media="(max-width: 1200px)" srcset="tablet.jpg">
    <img src="desktop.jpg" alt="艺术指导图片">
</picture>

💡 性能提示

使用 loading="lazy" 实现原生懒加载，减少初始页面加载时间。

2. 现代图片格式支持

<picture>
    <source type="image/avif" srcset="image.avif">
    <source type="image/webp" srcset="image.webp">
    <img src="image.jpg" alt="渐进增强的图片">
</picture>

格式对比

AVIF: 最新格式，压缩率最高

WebP: 较好的压缩，广泛支持

JPEG: 传统格式，全面支持

6. 媒体性能优化建议

🚀 图片优化

  • 使用适当的图片格式（AVIF > WebP > JPEG/PNG）
  • 提供多种尺寸，让浏览器选择最合适的
  • 使用懒加载减少初始加载
  • 设置明确的宽高避免布局偏移
  • 使用CDN加速图片加载

🎬 视频优化

  • 使用 preload="metadata" 只加载元数据
  • 提供多种质量选项
  • 移动端使用 playsinline
  • 考虑使用视频流服务（如HLS）
  • 添加合适的 poster 图片

📱 移动端优化

  • 检测网络类型，调整媒体质量
  • 使用 media 属性提供不同设备的资源
  • 考虑数据节省模式
  • 提供下载选项供离线查看
